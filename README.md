# 🚀 CampusSync

**CampusSync** is a modern **MERN-based campus platform** designed to simplify campus life by combining multiple useful services into one powerful system..

The platform provides **Lost & Found management, a student marketplace, AI-powered study tools, and an AI interview practice system**. CampusSync helps students connect, trade, learn, and prepare for careers using modern web technologies and artificial intelligence.

---

# 🌟 Key Features

## 🧑‍🎓 AI Interview Practice

CampusSync includes a smart **AI Interview System** that helps students prepare for real-world interviews.

Features:

* Select **job role, experience level, and interview mode**
* **AI interviewer asks questions using voice**
* **Speech recognition for answering**
* **Timer-based interview simulation**
* **Resume upload and AI resume analysis**
* **Detailed performance report**

The system evaluates users based on:

* Confidence
* Communication
* Correctness

After the interview, a **complete report is generated with feedback and scores**.

---

## 📚 AI Exam Notes Generator

Students can generate **AI-powered study notes** instantly.

Features:

* Topic-based notes generation
* Automatic **diagrams and charts**
* **PDF export support**
* Clean and structured learning content
* Credit-based generation system

This feature helps students **quickly understand complex topics**.

---

## 🛒 Campus Marketplace

Students can **buy and sell items within the campus community**.

Features:

* Post products
* Browse marketplace listings
* Contact sellers
* Simple campus trading system

---

## 🔍 Lost & Found System

CampusSync allows students to **report and find lost items easily**.

Features:

* Post lost items
* Report found items
* Search listings
* Contact item owners

This helps students recover items quickly and efficiently.
